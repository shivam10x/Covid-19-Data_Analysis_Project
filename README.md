#  COVID-19 Data Analysis

## 📌 Overview

This project analyzes COVID-19 data to understand trends in confirmed cases, deaths, and recoveries across different countries.

---

## 🎯 Objectives

* Track global COVID-19 trends
* Identify most affected countries
* Analyze death and recovery rates

---

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 📊 Dataset

* COVID-19 Global Dataset (CSV format)

---

## 📈 Key Insights

* USA and India reported the highest cases
* Death rates decreased over time
* Recovery rates improved

---

## 🚀 Conclusion

The project helps understand the spread and impact of COVID-19 using data analysis and visualization.
